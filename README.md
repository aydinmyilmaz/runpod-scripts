# RunPod Deployment Scripts

Scripts for deploying Chatterbox Multilingual TTS on RunPod.

## Usage

### Gradio App Deployment

```bash
# Clone this repo
cd /workspace
git clone https://github.com/aydinmyilmaz/runpod-scripts.git
cd runpod-scripts

# Run the deployment script
bash deploy_gradio.sh
```

## Scripts

- `deploy_gradio.sh` - Deploys the Gradio app version of Chatterbox Multilingual TTS on RunPod
